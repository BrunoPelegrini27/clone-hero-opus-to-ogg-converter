# Conversor de .opus para .ogg para Clone Hero

**Problema comum:** Músicas baixadas do [Enchor.nós](https://www.enchor.us) vem em .opus, e o Clone Hero nem sempre toca direito (pula pro fim ou não carga áudio). Este script converte pra .ogg (formato compatível) e substitui os arquivos originais.

## Como Usar (Windows)

1. **Instalar o Python:** Baixo em [python.org](https://www.python.org/downloads/) (marca "Adicionar ao PATH").

2. **Instalar o FFmpeg:** Baixo o ZIP em [gyan.dev/ffmpeg](https://www.gyan.dev/ffmpeg/builds/), extraia pra `C:\ffmpeg`, adicione `C:\ffmpeg\bin` ao PATH (Configurações > Sistema > Sobre > Configurações avançadas > Variáveis de ambiente).

3. **Baixo este roteiro:** Clique no botão verde "Código > Baixar ZIP", extraia.

4. **Roda:** Abra o Prompt de Comando (Win+R > cmd), vá pra massa do script (`downloads de cd\clone-hero-opus-to-ogg-converter-main`), digite `conversor Python_opus_para_ogg.py`.

5. **Selecione um macarrão raiz** (ex: sua massa "músicas" com subpastas de músicas).

6. **Aguarde:** Ele processa subpasta por subpasta. Backup antes!

**Testado no Windows 11.** Se der erro, confira o PATH ou mande issue aqui.

How to Use (Windows)

Download this script: Click the green "Code > Download ZIP" button, then extract.
Run: Open Command Prompt (Win+R > cmd), navigate to the script folder (cd Downloads\clone-hero-opus-to-ogg-converter-main), and type python converter_opus_para_ogg.py.
Select the root folder (e.g., your "songs" folder with music subfolders).
Wait: It processes subfolder by subfolder. Backup first!
